# Golf Score

這是可直接部署到 GitHub Pages 的版本。

## 上傳方式
1. 在 GitHub 建立一個新的 repository，例如 `golf-score`
2. 把這個資料夾內的所有檔案上傳到 repository 根目錄
3. 到 repository 的 **Settings → Pages**
4. 在 **Build and deployment** 裡把 **Source** 設成 **Deploy from a branch**
5. Branch 選 **main**，Folder 選 **/(root)**
6. 存檔後等待 GitHub Pages 發布

## 進站網址
發佈後網址通常會是：

`https://你的帳號.github.io/golf-score/`

## 手機加入主畫面
1. 用 Safari 打開上面的網址
2. 點分享
3. 選 **加入主畫面**

## 備註
- 球場與設定資料會存在瀏覽器的 localStorage
- 換手機或清除 Safari 網站資料後，已存球場資料不會跟著同步
